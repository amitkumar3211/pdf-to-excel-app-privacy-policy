# Privacy Policy for PDF to Excel Converter

Last updated: 2026

PDF to Excel Converter is developed by Amit Kumar.

This app allows users to convert PDF files into Excel format directly on their device.

## Information Collection

The app does NOT collect, store, or share any personal information.

## File Access

The application may request access to storage only to allow users to select PDF files for conversion. These files remain on the user's device and are not uploaded to any server.

## Data Security

All file processing happens locally on the user's device. We do not store or transmit any user files.

## Third-Party Services

This app does not use third-party services that collect personal information.

## Children's Information

The app does not knowingly collect personal information from children under 13.
